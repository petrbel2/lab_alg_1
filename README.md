Алгоритм Гаусса без выбора ведущего элемента:
В цикле для каждой строки i матрицы кроме последней, из каждого элемента нижележащих строк j вычитается элемент i строки того же столбца домноженный на фактор(элемент[j][i] / элемент[i][i]). Из правых частей вычитается i-ная правая часть домноженная на фактор. Таким образом, все элементы i-тых столбцов строк ниже i-ной обнуляются. В результате возникает верхнетреугольная матрица.

Обратная подстановка позволяет вычислить x.
